Each colored line is one source's own median (Windy = 2 models, Poseidon = 1 high-res Aegean model, days 1–5 only). Windy series are lightly smoothed (centered ±3 h weighted average, plus a light ±1 h pass on wave height) to damp step-to-step run noise; Poseidon is shown raw. Red stripes + ⚠ mark hours where the two sources disagree by ≥ 6 kt wind or ≥ 0.5 m wave — the days to re-check before committing. Grey band = min–max across all models; table numbers = all-model consensus (median). Horizons: Windy GFS ≈ 10 d, ICON-EU ≈ 5 d, Poseidon ≈ 5 d — beyond day 5 the forecast rests on Windy alone (and the wave line on a single model), so treat late days as tendency, not forecast. Agreement = band width vs median (tight ≤ 25 %, fair ≤ 50 %, poor beyond). Data: Windy Point Forecast API (personal key embedded in this file; quota-limited) + Poseidon HCMR (via public CORS relay, so it can occasionally be slow or unavailable — the page then continues without it; wave grid masked very close to shore). All times GMT+3 (Europe/Istanbul, no DST): both sources' UTC timestamps are converted onto the same hourly grid, so every chart hour and table day means the same wall-clock moment for both sources.
